### 控制台命令

控制台页面支持以下命令：

| 命令 | 功能 |
|------|------|
| `restart` | 重启程序 |
| `exit` / `quit` | 退出程序 |
| `clear` | 清空控制台显示 |
| `reload` | 重新加载日志文件 |
| `info` | 显示程序信息（Python版本、窗口大小、DPI缩放、背景设置等） |
| `help` | 显示所有可用指令的帮助信息 |
